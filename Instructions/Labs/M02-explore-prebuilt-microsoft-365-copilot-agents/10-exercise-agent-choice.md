Microsoft 365 Copilot propose désormais un écosystème toujours plus riche d’assistants spécialisés basés sur l’intelligence artificielle. Cette diversité vous donne la liberté de choisir l’outil le plus pertinent pour chaque tâche. Pour vous aider à préparer vos réunions, gérer vos projets, rattraper les communications manquées ou analyser des données, des assistants prédéfinis utilisent vos propres données Microsoft 365 pour simplifier votre flux de travail. Lors de cette activité, vous avez l’opportunité d’explorer librement l’un de ces assistants, tester ses fonctionnalités et l’appliquer à un véritable défi de productivité issu de votre travail quotidien.

> [!NOTE]
> Ce cours n'inclut pas un laboratoire Microsoft 365 avec des données fictives. En revanche, vous devez effectuer cet exercice de formation à l’aide de vos propres données personnelles. 

### Exercice

Cet exercice pratique vous permet d’adopter une approche concrète pour comprendre le fonctionnement des assistants prédéfinis de Microsoft 365 Copilot Chat, en l’appliquant à une situation réelle de votre travail récent. Revenez à un projet sur lequel vous avez travaillé au cours des 90 derniers jours, quelque chose qui nécessitait de la coordination, de la communication ou du suivi. Peut-être s’agissait-il d’un rapport à rédiger, d’un projet d’équipe qui a pris du retard, d’une requête client qui a disparu dans vos e-mails, ou d’une série de réunions où les décisions n’ont jamais été clairement consignées.

Votre tâche est de choisir un assistant prédéfini de Microsoft 365 Copilot Chat que vous pensez capable de vous aider à résoudre ou à obtenir des aperçus pour faire avancer votre projet. 

> [!IMPORTANT]
> Les exercices précédents dans ce module ont utilisé les assistants Analyste et Recherche. Pour cet exercice, évitez de réutiliser ces assistants. Choisissez plutôt un autre assistant prédéfini pour vous assister sur le projet que vous avez sélectionné. Cette approche garantit que vous acquérez une expérience de l’éventail complet des capacités de Copilot et comprenez quel assistant est le plus approprié pour différents types de tâches.

Une fois l’assistant sélectionné, explorez ses fonctionnalités à l’aide de vos propres données Microsoft 365, comme des e-mails, des réunions, des fichiers ou des fils de conversations. L’objectif est de voir si vous pouvez faire progresser le projet ou découvrir quelque chose que vous auriez pu manquer. À la fin de cet exercice, vous serez également invité à tester d’autres assistants prédéfinis pour observer les types de requêtes qu’ils peuvent traiter.

Suivez les étapes ci-dessous pour découvrir comment l’assistant prédéfini que vous avez choisi peut réellement devenir un partenaire de productivité en résolvant un problème qui vous intéresse :

1. Dans **Microsoft Edge**, ouvrez un nouvel onglet et entrez l’URL suivante : [**https://M365copilot.com**](https://M365copilot.com)
1. Dans **Microsoft 365**, sélectionnez **Tous les assistants** dans le volet de navigation, puis, dans la fenêtre **Magasin d’assistants**, sélectionnez l’assistant de votre choix dans la section **Définis par Microsoft**. N’oubliez pas, n’utilisez pas les assistants Recherche ou Analyste. Testez un assistant différent pour cet exercice. 
1. L’objectif de cet exercice est d’interagir avec l’assistant en utilisant vos propres données. Commencez par examiner les suggestions d’invite intégrées proposées par l’assistant. Essayez une ou deux des exemples d’invites affichées sur la page de l’assistant. 
1. Après avoir testé les exemples d’invites, exécutez trois à cinq requêtes ou actions que l’assistant prend en charge. Ce faisant, explorez les données auxquelles l’assistant peut accéder, comme votre calendrier, e-mails, fichiers ou données organisationnelles. Les invites que vous entrez doivent évidemment correspondre à votre projet et à l’assistant que vous avez choisi. Toutefois, voici quelques exemples d’invites qui peuvent stimuler une idée du type d’invite que vous pouvez entrer :
   - Analysez une activité d’équipe (si l’assistant le prend en charge).
   - Identifiez les fichiers clés, les tâches ou les tendances.
   - Demandez comment intégrer un nouveau système.
   - Aidez à la réflexion pour un projet à venir.
   - Donnez des conseils pour faire progresser votre carrière.
   - Fournissez des commentaires sur un projet d’écriture.
1. Demandez maintenant à votre assistant de créer un résultat concret, comme un résumé, une liste, un document ou un plan d'action. Par exemple :
   - Passez en revue les messages manqués et générez un plan de suivi.
   - Rédigez un document de mise à jour de statut ou de préparation de réunion.
   - Créez un rapport qui récapitule le mode Conversation ou les conversations.
1. Profitez de cette opportunité pour tester d’autres assistants prédéfinis. D’abord, passez en revue les exemples d’invites qu’ils fournissent, indicatives des types de requêtes qu'ils spécialisent. Exécutez quelques-uns de ces exemples d’invites ou entrez vos propres invites personnalisées pour vous familiariser avec les types de demandes que d’autres assistants peuvent prendre en charge.
